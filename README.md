# 💰 Expense Tracker

A full-stack web application to track income, expenses, and financial habits with real-time insights.

---

## 🚀 Features

* ➕ Add income & expenses
* 📊 Track balance in real-time
* 🗂️ Categorize transactions
* 📈 Visualize spending patterns *(upcoming)*
* 🧾 Transaction history
* ❌ Delete or edit entries

---

## 🏗️ Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS
* Axios

### Backend *(if added / planned)*

* Spring Boot / Node.js

### Database *(if added / planned)*

* MySQL / MongoDB
